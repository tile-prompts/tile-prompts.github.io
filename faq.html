FAQ

Frequently Asked Questions

Quick answers to the common stuff: how tiles work, how the draw-pile and rerolls behave, what “sets” are, etc.

What is this site, exactly? Basics ▼

Tile-Prompts is a static, no-login prompt generator. You build a prompt by drawing tiles (Roles, Traits, Situations, Place, and Task).

Think “prompt lego's”: snap together a few character role and trait tiles, then add a shared situations, places, and tasks.

Why can I draw multiple Roles even though it doesn't always make sense Rules ▼

Traits are character-specific, and Situations are “global modifiers” that stack nicely. Place and Task act more like single anchors: one setting and one win-condition keeps the prompt cohesive. Roles are a bit of an anomaly, they can sometimes fit well together, so I left the ability to have more than one. This allows a character to be "A Wizard" and "A Dragon" at the same time. Sometimes it doesn't work at all though, and you get something like "A Single Sugar Ant" and "Your Middle-School Janitor".

Feel free to contact me if you have ideas or suggestions relating to this issue.

What does the “Sets” tab do? Content ▼

Sets are content packs. Turning a set on adds its entries into the active pool for that tile type; turning it off removes them from future draws.

If you’re looking for a specific vibe, toggling sets is the fastest way to steer the generator.

Why is a button disabled sometimes? UI ▼

Usually it means there isn’t enough content in the relevant draw pile to complete the action safely (especially rerolls), or a feature is intentionally not available yet.

If you disabled most sets or tiers, try re-enabling them to refill the available pool.

I’m “out of tiles.” How do I reset things? Troubleshooting ▼

Use Clear All Tiles in the Setup menu. That clears placed tiles and refills the draw piles based on your enabled sets and tier filters.

If you want more variety, enable additional sets.

Can I add my own sets? Custom ▼

Yep. This project is designed around text-based sets. If you wish to contribute on the GitHub, the simplest path is to add a new .set file in the appropriate folder and include it in that folder’s index.json.

Fastest way to get a good prompt

Create 2–3 characters total, give each character 1 Role, then add 1–2 Traits per character, add a 1-2 Situations, then lock in Place and Task last. For simpler prompts only do one thing at a time in the meta panel.
